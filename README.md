# bedwars-pack-for-pvp-1.12-
A basic resource pack for 1.12 PvP players :D

## Resource pack

A minimal Minecraft Java Edition **1.12** resource pack was added to this repository.

[![Release](https://img.shields.io/github/v/release/lumin0sity-dev/bedwars-pack-for-pvp-1.12-?style=flat)](https://github.com/lumin0sity-dev/bedwars-pack-for-pvp-1.12-/releases/latest)

- Build script: `create_pack.sh` (generates `bedwars-pvp-1.12-resourcepack.zip`)
- Pack metadata: `resourcepack/pack.mcmeta` (uses `pack_format: 3` for 1.11â€“1.12)

To create the downloadable ZIP run:

```bash
bash create_pack.sh
```

Or download the latest release ZIP directly from GitHub:

- Latest release: https://github.com/lumin0sity-dev/bedwars-pack-for-pvp-1.12-/releases/latest
- ZIP (v1.0.0): https://github.com/lumin0sity-dev/bedwars-pack-for-pvp-1.12-/releases/download/v1.0.0/bedwars-pvp-1.12-resourcepack.zip

Then move `bedwars-pvp-1.12-resourcepack.zip` to your Minecraft `resourcepacks` folder and enable it in the in-game Resource Packs menu.
